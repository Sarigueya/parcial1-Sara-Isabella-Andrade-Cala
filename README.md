# 📝 Parcial 1 — Programación Web

Proyecto evaluativo del primer parcial del curso de **Programación Web**. Aplica conceptos de maquetación, diseño responsivo y validación de formularios con JavaScript.

---

## 📋 Contenido

- Estructura semántica con **HTML5**
- Diseño responsivo con **Bootstrap** y **CSS**
- Validación de campos con **JavaScript** y expresiones regulares
